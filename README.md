# software
My sofware es una app para niños de 5 a 7 año el cual mediante imagenes y juegos se le enseña al niño para mejorar el conocimineto. 

Descargar:
npm create vite@latest kid-learn-app -- --template react-ts
npm install
powershell -NoProfile -Command "Start-Sleep -Seconds 5"
npm install vite-plugin-pwa
npm install -D vite@^7.0.0
npm view @vitejs/plugin-react@5 peerDependencies
npm install -D @vitejs/plugin-react@5.1.4
npm install -D vite-plugin-pwa
npm audit fix
npm list vite-plugin-pwa
npm audit fix --force
npm view @vitejs/plugin-react@4 peerDependencies
npm install -D vite@^5.4.0 @vitejs/plugin-react@^4.7.0 vite-plugin-pwa@1.2.0
